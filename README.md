# Poke Repeat - 戳一戳重复最新消息插件

[![AstrBot](https://img.shields.io/badge/AstrBot-v3.4%2B-blue)](https://github.com/Soulter/AstrBot)

## 📖 简介

这是一个为 [AstrBot](https://github.com/Soulter/AstrBot) 开发的插件。当有人在群聊中戳机器人时，机器人会自动回复该群的最新一条消息。

### 功能特点

- ✅ **自动缓存**：自动记录每个群的最新消息
- ✅ **智能回复**：被戳时回复最后一条消息内容
- ✅ **忽略命令**：自动忽略以 `/` 开头的命令消息
- ✅ **多群隔离**：不同群的消息缓存互不干扰
- ✅ **简单易用**：无需配置，开箱即用
